Comecei na área de design gráfico, estudei na ESPM em 2012, sempre tive interesse por áreas que envolviam a criatividade, comunicação visual e tecnologia. Até que em 2017, conheci a área de ux e ui, me envolvi, fiz diversos cursos como o Tera, desde então me apaixonei pela área. Sempre com novos desafios, tentando resolver os problemas de uma forma mais humana, com o design, a tecnologia e muita pesquisa.

O que eu faço?

Tenho expertise em user experience assim como user interface, então costumo trabalhar de ponta a ponta, ou seja, desde a etapa do descobrimento até a prototipacão do produto. O que posso fazer:

User Experience (UX)
• Pesquisa quantitativa
• Pesquisa qualitativa
• Coleta de dados
• Persona
• Fluxo/jornada do usuário
• Fluxo/jornada sistêmico
• Arquitetura da informação
• Teste de usabilidade
• Análise heurística
Já desenvolvi o design System das empresas Gol linhas aéreas e para o banco Itaú IBBA. Como user interface posso:
User Interface (UI)
• Prototipos navegáveis (wireframe, alta)
• Desenvolvimento do design system (documentação, componentização)
• Documentação para desenvolvedores
